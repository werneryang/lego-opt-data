# 策略分析框架 v2（特征与风险扩展）

## 目标
在 v1 基础上扩展因子体系、风险管理与策略稳健性评估，使框架可支持多信号组合与初步风控。

## 数据融合与特征扩展
- **Streaming → 微观结构特征**
  - 成交方向不平衡（tick imbalance）。
  - 盘口深度变化速率。
- **5 分钟 → 技术指标**
  - ATR、RSI、MACD（短周期）。
  - 动量与反转组合特征。
- **EOD → 风险与宏观**
  - 日级 realized vol、隔夜 gap。
  - 简化风险因子（beta、相关性、波动结构）。

## 风险控制
- **仓位控制**：基于波动率目标（vol targeting）。
- **风控约束**：最大回撤阈值、单日损失阈值、持仓集中度限制。

## 评估扩展
- **稳定性**：滚动窗口评估、参数敏感度测试。
- **收益指标**：Sharpe、Sortino、Calmar。
- **风险暴露**：对市场因子、波动因子的回归分解。

## 输出与落地
- **特征表**：新增 `features_microstructure` 与 `features_risk`。
- **信号组合**：多信号线性/非线性组合评分。
- **评估报告**：`reports/v2`，包含风险与稳健性分析。

## 版本边界（v2）
- 支持单标的（SPY）多信号组合。
- 支持简单交易成本（固定点差 + 滑点）。
- 暂不支持多标的横截面。
