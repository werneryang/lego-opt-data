# 基于 SPY streaming / 5 分钟 snapshot / EOD 的期权策略与使用方法

## 可用数据回顾
- **Streaming**：逐笔/盘口级数据，用于捕捉微观结构变化（成交不平衡、盘口厚度、瞬时波动）。
- **5 分钟 snapshot**：稳定的区间级指标，用于构建可回测的日内信号与交易路径。
- **EOD**：日级趋势与风险因子，用于中期方向、波动结构与风险控制。

## 策略类型与使用方法

### 1) 方向性策略（趋势 / 反转）
**适用数据**：5 分钟 snapshot + EOD
- **趋势跟随**：
  - 使用 5 分钟动量（短周期均线/突破）与 EOD 趋势确认。
  - 触发时构建方向性期权仓位（如买入看涨/看跌、或合成期货）。
- **均值回归**：
  - 识别 5 分钟异常超跌/超涨（z-score、RSI）并结合 EOD 波动水平控制仓位。
  - 通过短期限价差（debit spread）降低成本。

### 2) 波动率策略（多/空波动）
**适用数据**：5 分钟 snapshot + EOD + Streaming
- **短期波动率交易**：
  - 5 分钟 realized vol 与 EOD 波动结构对比，识别波动溢价/折价。
  - streaming 监控波动突变触发时机；适合跨式/宽跨式（straddle/strangle）。
- **波动率回归**：
  - 日内 realized vol 偏离长期均值时，通过 calendar spread 或 ratio spread 进行波动结构交易。

### 3) 事件驱动策略
**适用数据**：Streaming + 5 分钟 snapshot
- **事件冲击捕捉**：
  - 当 streaming 显示成交量与盘口厚度异常，触发短期趋势或波动策略。
  - 以 5 分钟桶进行信号稳定性过滤，避免噪声。

### 4) 结构化收益策略（收益增强 / 风险对冲）
**适用数据**：EOD + 5 分钟 snapshot
- **备兑开仓（Covered Call）**：
  - EOD 趋势偏多时卖出短期限价差以获取权利金。
  - 5 分钟波动高位时择机卖出以提升权利金。
- **保护性认沽（Protective Put）**：
  - EOD 风险指标上升时建立保护性仓位。

### 5) Gamma Scalping（短期波动捕捉）
**适用数据**：Streaming + 5 分钟 snapshot
- 在 streaming 中识别高频波动与盘口变化，结合 5 分钟波动指标。
- 通过期权多头（长 gamma）进行日内对冲交易，利用价格波动获利。

## 数据使用流程（建议）
1. **Streaming → 微观结构特征**：
   - 盘口厚度、成交不平衡、瞬时波动。
2. **5 分钟 snapshot → 可回测信号**：
   - 动量/反转、区间波动、成交量放大。
3. **EOD → 风险过滤与仓位管理**：
   - 趋势方向、长期波动结构、风险预算。

## 输出建议
- **策略信号表**：各策略独立信号 + 组合信号。
- **交易回测报告**：成本、滑点、PnL 分解。
- **风险监控指标**：波动暴露、最大回撤、收益稳定性。
