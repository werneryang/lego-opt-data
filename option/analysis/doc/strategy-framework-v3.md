# 策略分析框架 v3（组合与执行模拟）

## 目标
在 v2 基础上加入组合层与执行模拟，构建更贴近实盘的策略评估框架。

## 组合与信号融合
- **信号融合**：
  - 线性权重（IC/IR 加权）。
  - 规则集（趋势 + 波动 + 流动性条件）。
- **组合层**：
  - 目标风险敞口与仓位分配。
  - 时间分层（盘中与隔夜信号分离）。

## 执行模拟与交易成本
- **执行模型**：
  - 基于 5 分钟桶的成交模拟（VWAP/中间价成交）。
  - 盘口宽度驱动的动态滑点。
- **成本分解**：
  - 显性成本：点差、佣金。
  - 隐性成本：冲击成本估算。

## 回测框架增强
- **事件回测**：围绕高波动事件（FOMC、CPI）建立子样本评估。
- **分区评估**：不同市场状态（高波动/低波动）下的表现对比。
- **稳定性与退化监控**：信号失效检测与漂移指标。

## 输出与落地
- **组合层输出**：`portfolio_5m`（仓位、风险暴露、PnL）。
- **执行模拟报告**：`reports/v3`（成本、成交质量）。

## 版本边界（v3）
- 支持单标的组合策略。
- 引入执行模拟，但仍以 5 分钟粒度为主。
