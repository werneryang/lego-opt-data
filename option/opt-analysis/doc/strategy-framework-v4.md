# 策略分析框架 v4（监控与生产化）

## 目标
在 v3 基础上补全生产化与监控流程，形成可持续运行的策略分析与迭代体系。

## 生产化流程
- **数据质量监控**
  - streaming 丢包率、snapshot 延迟、EOD 对账完整性。
  - 异常报警：缺失数据、异常波动、成交量突变。
- **策略运行监控**
  - 实时表现 vs 历史基线偏离。
  - 关键风险指标：日内 VaR、回撤、仓位偏移。

## 评估与复盘机制
- **日内复盘**：盘中信号触发与执行质量评估。
- **日终复盘**：PnL 分解、风险暴露归因。
- **策略退化处理**：触发降级/停用规则与策略再训练门槛。

## 迭代与治理
- **版本控制**：策略版本元数据（参数、因子、训练数据窗口）。
- **实验治理**：A/B 测试、影子策略运行。
- **审计与留档**：关键变更记录、审计日志。

## 输出与落地
- **监控指标表**：`monitoring_metrics`。
- **运行报告**：`reports/v4`（日内/日终监控与异常事件）。
- **治理文档**：策略版本清单与发布记录。

## 版本边界（v4）
- 支持持续运行与迭代。
- 暂不涉及多标的扩展，但接口保留可扩展性。
